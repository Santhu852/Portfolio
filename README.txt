Santhosh Kumar — Portfolio (Static HTML/CSS)
=============================================

This is a fully static HTML + CSS version of the website.
No build tools, no JavaScript framework required.

Pages:
  - index.html      Home / About
  - work.html       Portfolio
  - services.html   Services
  - resume.html     Resume & skills
  - contact.html    Contact form

How to view:
  Just open index.html in a browser.
  Or serve the folder with any static server, e.g.:
    python3 -m http.server 8080

Folder structure:
  index.html, work.html, services.html, resume.html, contact.html
  styles.css
  assets/   (all images)
